Shipping Rates Cities Woocommerce

== Description ==

Este plugin permite definir precios de costos de envío por ciudad, podrá definir las diferentes tarifas editando el excel y subiendo en las configuraciones del plugin
[Descargar excel base](https://drive.google.com/file/d/1km4dti3Qkwk2PIbzMJWpS95L4fChsJXL/view?usp=sharing)

== Installation ==

1. Ingrese a su administrador de wordpress
2. Ingrese al menu plugins / añadir nuevo
3. Busque por el nombre **Shipping Rates Cities Woocommerce**
4. Instale y active el plugin y siga el proceso de configuración.
5. Establezca los detalles del envío del producto, ver captura de pantalla.
